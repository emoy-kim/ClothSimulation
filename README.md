## Keyboard Commands
  * **Up Arrow**: move forward
  * **Down Arrow**: move backward
  * **Left Arrow**: move left
  * **Right Arrow**: move right
  * **w key**: move up
  * **s key**: move down
  * **i key**: main camera and projector reset
  * **l key**: light turn on/off
  * **enter key**: project an image/video
  * **q/ESC key**: exit
